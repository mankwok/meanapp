## Installation
1. Clone the repo and install dependencies of API server:
  ```
  $ git clone https://github.com/mankwok/meanapp.git
  $ cd meanapp
  $ npm install
  ```
2. Install dependencies of Angular application:
```
$ cd angular-app/
$ npm install
```

## To Run The Application
- For development:
1. Run API Server:
```
$ npm run devstart
```
2. Run Angular Application:
```
$ cd angular-app/
$ ng serve
```
